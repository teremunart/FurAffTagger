# Tagging...
...on FurAffinity is a nightmare!
This should ease the pain a lil bit.

It is definitely not perfect but makes life already better when you are an artist that uploads on FurAffinity.

---

It uses a CSV from DraconicDragon Repo that scraps the e621 one, but works like a charm.
https://raw.githubusercontent.com/DraconicDragon/dbr-e621-lists-archive/refs/heads/main/tag-lists/e621/e621_2026-01-01_pt20-ia-ed.csv

# Quick Installation
since at the moment it is not available in the Chrome Store.

1. Download your extension and unzip it. Make sure you can see a file named manifest.json inside the folder.
2. Open Chrome and type chrome://extensions/ in the address bar, then hit Enter.
3. Switch the Developer mode toggle in the top-right corner to ON.
4. Click the Load unpacked button that appears. Select the folder containing your files and click Select Folder.
